# PROJECT.md

## 프로젝트명
세브란스 보호자 교육 포털 (Parents Education Portal)

## 설명
상급종합병원 보호자를 위한 환자 교육 웹 애플리케이션.
현재 NICU(신생아집중치료실) 전용으로 운영 중이며, 다중 진료과 지원 플랫폼으로 확장 진행.

## 기술 스택
- Frontend: React 19 + TypeScript + Tailwind CSS v4 + Vite
- Backend: Express.js + sql.js (in-memory SQLite)
- AI: Google Gemini 2.0 Flash (챗봇)
- 배포: Vercel (serverless)
- 인증: JWT (보호자: 등록번호, 관리자: 이메일/비밀번호)

## 현재 상태
- v1: NICU 전용 버전 운영 중
- 주요 기능: 입원안내, 치료과정, 퇴원안내, 외래안내, 동영상 교육, Q&A 챗봇, 바이탈 모니터링, 퇴원 매뉴얼
- 관리자: 콘텐츠 CRUD, 알림 발송, 입원 확인

## 현재 마일스톤: v2 — 다중 진료과 확장

### 목표
NICU에 국한된 앱을 범용 보호자 교육 플랫폼으로 확장.
3개 진료과 버전 추가: 정형외과 병동, 심혈관 중환자실(CCU), 소아응급실.

### 핵심 요구사항
1. **멀티 디파트먼트 아키텍처**: 진료과(department) 개념 도입, DB/라우팅/인증에 반영
2. **진료과 전환 UI**: 랜딩페이지에서 진료과 선택 → 해당 전용 대시보드 진입
3. **진료과별 콘텐츠 분리**: 각 진료과 고유 카테고리, 카드, 케어저니, 퇴원매뉴얼
4. **관리자 다중 진료과 관리**: 진료과별 독립 콘텐츠 관리
5. **기존 NICU 버전 유지**: 현재 NICU 콘텐츠를 department='nicu'로 마이그레이션

### 제약사항
- sql.js in-memory DB 유지 (Vercel 호환)
- 기존 보호자 UX 최소 변경
- 모바일 우선 반응형 유지
